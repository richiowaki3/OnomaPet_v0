# OnomaPet 00

`OnomaDict`（オノマトペ辞書）の16次元ベクトル（ラバン・エフォート、音響物理、拡張感覚、フレージング拍節）を利用して、3D空間上の4ノードの面を動かす簡単な可視化デモです。

## 構成ファイル / File Structure
- `index.html` : ユーザーインターフェースとスタイリング（ガラスモーフィズム、ダークテーマ）。
- `app.js` : Three.jsによるシーン初期化、描画ループ、物理・運動変換マッピングロジック。
- `onomatopoeia_dictionary.js` : `OnomaDict`のデータ（764語）をローカル環境で直接ロードできるようJavaScript変数に変換した辞書データ。
- `convert_dict.py` : 元のJSON形式の辞書から `onomatopoeia_dictionary.js` を再ビルドするためのPythonスクリプト。

## 運動マッピングのルール / Motion Mapping Rules
各単語のパラメータ（0〜9）に基づいて、4つのノード（頂点）とそれを結ぶ面（Plane）の運動特性が変化します。

1. **重さ / Weight (x1)**
   - 全体の振幅を大きくします。また、値が大きいほど重力による「たわみ（Sag）」が発生し、面が全体的に下方向（Y軸マイナス）に引っ張られます。
2. **時間への態度 / Time (x2) & 減衰 / Decay (x8)**
   - **Sustained（値が小さい）**：なめらかな正弦波（sin）状の往復運動になります。
   - **Sudden（値が大きい）**：急激なインパルス（打撃）の後に指数関数的（exponential）に減衰する動きになります。Decay (x8) が高いほど、減衰がより速くなります。
3. **空間への態度 / Space (x3)**
   - **Direct（値が大きい）**：ノードは直進的（Y軸方向のみ）に上下します。
   - **Indirect（値が小さい）**：ノードは3D空間内を円状、またはフィギュア8（8の字）状に軌道を描く（Orbital）複雑な進路を取ります。
4. **流動性 / Flow (x4)**
   - 抑制（Bound、値が大きい）されるほど、摩擦とインピーダンスを表現するために全体の振幅が制限されます。
5. **硬さ / Hardness (x5)**
   - **Rigid（値が大きい）**：4つのノードが完全に同期（同位相）して動き、面は折れ曲がらず硬い板のように並進・傾き運動をします。
   - **Fluid（値が小さい）**：ノード間に時間的・位相的な遅れ（Phase Offset）が発生し、ゼリーや波のようにうねる（Wavy）動きを表現します。
6. **拍節頻度 / Meter (x15) & 規則性 / Regularity (x16)**
   - Meterが高いほど、1ループ内の運動速度（周波数）が速くなり、またサブインパルス（反復）が発生します。
   - Regularity（不規則、値が大きい）ほど、時間経過に微細なジッターが加わり、カクついた不揃いな運動になります。
7. **乱流 / Reynolds (x9)**
   - 値が大きいほど、ノード位置に対して高周波の細かなノイズ・震え（Turbulence）が加算されます。

8. **触色 / Color (x12)**
   - 辞書に設定された各単語の `color_hex`値 を読み込み、ノードおよび面の色がリアルタイムでその色に変わります。

## 実行方法 / How to Run
1. `onomapet00` フォルダ内の `index.html` ファイルをブラウザでダブルクリックして開きます。
   - **CORS制限なし**: 辞書データを `onomatopoeia_dictionary.js` として直接ロードするため、Webサーバーを立ち上げることなくローカルのままで動作します。
2. 画面左上の入力欄にオノマトペ（例：「ふわふわ」「あたふた」「がたがた」など）を入力します。入力欄をクリックすると、登録されている単語のリストが表示されます。
3. 「**実行**」ボタンを押す（またはEnterキーを押す）と、4つのノードが瞬時に原点（静止位置）にリセットされた後、新しい言葉の運動ループを開始します。
4. 「**ランダム**」ボタンで辞書からランダムにオノマトペを選んで実行することもできます。
5. 3Dビューポートはマウスやタッチ操作でドラッグ（回転）、スクロール（ズーム）、右ドラッグ（平行移動）が可能です。
